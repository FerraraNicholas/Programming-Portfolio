Health Tracker

A single page application that tracks user's calorie intake. Displays such functionality as typing food names into a search that will display a list matching foods provided by a health API. Users are able to select an item from the list and the item will be added to the list of foods the user is tracking. Calorie intake will be updates to reflect the new daily total.
